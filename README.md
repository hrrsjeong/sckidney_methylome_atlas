# Single-Cell DNA Methylome Pipeline (sciMETv2)

This repository contains the **Snakemake** pipeline used to process and analyze single-cell DNA methylation data (sciMETv2) generated from human and mouse kidneys.

## Acknowledgements

Portions of the preprocessing and demultiplexing logic in this pipeline were adapted from the original **Adey Lab** repository.

* **Original Repository:** [https://github.com/adeylab/sciMETv2](https://github.com/adeylab/sciMETv2)
